## 🚀 Featured Projects

<details>
<summary><b>💼 NSK CRM — Academic Branch Operations & Lifecycle Management Platform</b></summary>
<br>

### Project Description
A unified, production-grade enterprise software platform designed to manage the entire lifecycle of an academic training branch. Features lead & enquiry tracking, telephone follow-up logging, automated batch seat allocation gauges, monthly installment planning, offline Android client access, and an advanced in-memory preloading system for zero-latency dashboard operations.

| Metric | Details |
| :--- | :--- |
| **Stack** | Next.js 15, React 19, TypeScript, Tailwind CSS v4, Shadcn/UI, Recharts, PostgreSQL, Supabase, Expo (React Native) |
| **Scale** | Fully coordinates operations, fees, batches, and records for 100+ student classes and multiple corporate users |
| **Performance** | Instant 0ms client-side preloading caching and background database sync to prevent buffering spin delays |
| **Security** | Role-Based Access Control (RBAC) with 5 user roles (Admin, Manager, Counselor, Accountant, Trainer) & Audit Trail Logs |
| **Impact** | Completely digitalized daily operations for Networkz Systems Kollam Branch; eliminated receipt-processing friction |
| **Repository** | [nsk-crm](https://github.com/nahas2003/nsk-crm) |

### Professional Project Explanation
This system automates and coordinates branch-level training operations. Built with a Next.js 15 App Router backend leveraging Server Actions and a Supabase PostgreSQL instance, it features an interactive dashboard, Kanban lead trackers, follow-up planners, and an accountant module to configure installment schedules. Its mobile companion compiled via React Native enables mobile counseling staff to capture leads on the go. To address latency, the platform implements a lazy-loading cached dashboard strategy for zero-millisecond tab switching.
</details>

<details>
<summary><b>🌾 AI-Based Smart Farm Protection & Intrusion Detection System</b></summary>
<br>

### Project Description
An advanced AI-powered intrusion detection and deterrence system designed to protect agricultural farms from wildlife and bird intrusions. Built on the research published in the survey paper, this project leverages computer vision models to identify threat levels and trigger deterrence measures automatically.

| Metric | Details |
| :--- | :--- |
| **Stack** | Python, OpenCV, TensorFlow, Raspberry Pi, React.js Dashboard, MQTT |
| **Scale** | Real-time edge inference on IoT devices with centralized dashboard monitoring |
| **Performance** | 94.2% detection accuracy under varying lighting conditions with 150ms inference latency |
| **Security** | End-to-end encryption for IoT-to-cloud communications, secure device authentication |
| **Impact** | Winner of the Young Innovators Program (YIP) 8.0 (District Level), securing development funding |
| **Repository** | [smart-farm-protection](https://github.com/nahas2003/smart-farm-protection) |

### Professional Project Explanation
Agricultural systems lose significant yields annually to wildlife intrusions. This project implements a smart farm protection framework using edge-based computer vision. The system analyzes live camera feeds, classifies animal species using custom-trained light-weight TensorFlow models, and deploys context-aware non-lethal deterrence measures (e.g., specific ultrasonic frequencies or visual alerts). The centralized React dashboard provides farmers with real-time statistics, intrusion logs, and device health status, enabling proactive farm management.
</details>
